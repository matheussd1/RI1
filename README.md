#Vingadores#

##Atividade##
Um projeto, em JavaScript, para criar objetos querepresentem personagens do universo Marvel, especificamenteos vingadores.

##Objetivo##
Desenvolva um código que faça a comparação entre cada personagem, ou seja, seus atributos e por fim, seu projeto deve imprimir a descrição de cada personagem
que esta sendo comparado e também apontar onde um personagem “vence”
